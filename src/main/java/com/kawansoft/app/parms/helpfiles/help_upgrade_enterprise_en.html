Upgrade To The Enterprise Edition

Trying The Enterprise Edition

Transitioning from the default Community Edition to the Enterprise Edition simply requires a license file (there is no other installation to do):
  • Get a Trial License File at www.aceql.com/aceql-trial.
  • You'll receive an email with the license file.
  • Install the received aceql-license-key.txt license file in the
    <Installation Directory>/AceQL/lib-server/ subdirectory.
  • Restart the AceQL Server.

Buying The Enterprise Edition

When buying a license, the license is valid for one year and applies to up to 5 SQL database names.

There's no limitation to the number of computers installed. This means you're granted the right to access up to 5 databases from any computer.
The "my_company_sales" database, for example, may be accessed by AceQL from your test server, your pre-production server and your production server.
